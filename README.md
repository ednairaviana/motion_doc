# Motion Journal

reference: https://motion.dev/docs

A personal journal for studying motion and animation through small, focused code experiments.

This repository contains isolated studies, small experiments, and incremental explorations aimed at understanding motion principles on the web. The focus is on learning and developing intuition for timing, easing, spatial relationships, and interaction through practical implementation.

## Purpose

- Study motion and animation fundamentals
- Explore animation techniques through small, focused experiments
- Build intuition for timing, easing, and spatial relationships
- Document learning progress over time
